# Women Clothing Sales Analysis

## Overview
This project focuses on analyzing an e-commerce clothing dataset to understand sales performance, customer preferences, and revenue patterns.

## Tools
- Python
- Pandas
- Matplotlib

## Dataset
Women Clothing E-commerce Sales dataset.

## What I Did
- Cleaned and prepared the data
- Converted date columns and created monthly features
- Analyzed total and average revenue
- Identified top-performing products
- Explored color and size preferences
- Analyzed monthly sales trends

## Key Findings
- SKU 799 is the top revenue-generating product
- Sales peak in month 9
- Dark Blue products generate the highest revenue

## Conclusion
Sales are concentrated around specific products and time periods. These insights can help improve inventory and marketing decisions.
